# Coffee Sales Dashboard | Power BI

**Project Objective**
To analyze coffee shop sales using Power BI and provide insights on revenue trends, top-selling products, and operational performance.

**Key Features**
- 3 pages interactive dashboard: Executive Overview, Product Analysis, and Operations Insights
- KPIs: Total Sales, Transactions, Avg Ticket, Cash %, Top Products, and Forecast
- Custom coffee-themed layout and color palette
- DAX measures and time intelligence using the Date table

**Tools & Skills**
Power BI | Power Query | DAX | Data Modeling | Dashboard Design | Data Storytelling
